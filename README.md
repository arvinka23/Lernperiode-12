# Lernperiode-12

Technologien:
Backend: Golang (Gin oder Fiber Framework)
Frontend: React mit TypeScript
Datenbank: MongoDB
KI-Integration: OpenAI API (z. B. für automatische Filmbeschreibungen oder Empfehlungen)

7 Epics (User Stories)
Als Benutzer möchte ich mich registrieren und anmelden können, damit ich personalisierte Inhalte sehen kann.
Als Benutzer möchte ich Filme durchsuchen und Details ansehen können, damit ich passende Filme finde.
Als Benutzer möchte ich Filme abspielen können, damit ich sie direkt auf der Plattform ansehen kann.
Als Administrator möchte ich neue Filme hochladen und verwalten, damit ich das Angebot aktuell halten kann.
Als Benutzer möchte ich Empfehlungen basierend auf meinem Geschmack sehen, damit ich neue interessante Filme finde.
Als Benutzer möchte ich Kommentare und Bewertungen zu Filmen abgeben, damit ich meine Meinung teilen kann.
Als Entwickler möchte ich die App sicher, performant und ansprechend gestalten, damit sie professionell wirkt.
